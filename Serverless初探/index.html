操作场景

该任务指导您通过 Serverless Website 组件，快速构建一个 Serverless Hexo 站点。

前提条件

  • 已安装 Node.js（Node.js 版本需不低于8.6，建议使用 Node.js10.0 及以上版本）
  • 已安装 Git

如您未安装上述应用程序，可以参考 Hexo 安装说明。
